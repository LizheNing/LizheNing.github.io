我常用的办公软件

我比较喜欢尝试新的事物，喜欢用新的工具解决现存问题。因此经常把一些新的软件加入到工作流中。为了能在更换电脑之后迅速还原之前的工作模式，将常用软件记录在此。

即时通讯

  1. 应用名：微信
  2. 官网地址：微信官网可以下载
  3. 优点：用户基数庞大
  4. 缺点：垃圾中的战斗机，每次换手机电脑，都需要重新同步。最近发现电脑版微信增设了 “备份与恢复” 功能，可以实现手机到电脑和电脑到手机的部分聊天信息的双向恢复。
  5. 建议：工作上能不用微信就别用了，钉钉在文件传输，版本记录上强于微信很多。（没事用钉钉不是线的嘛！）。微信是一个获取信息，信息交换的重要通道。

远程控制

  1. 应用名：ToDesk
  2. 官网地址：ToDesk官网可以下载
  3. 优点：基本功能免费，很适合网络系统不完善的小公司。应对的场景主要是向打印机等主机传输文件。
  4. 缺点：用的功能比较少，暂时没有发现缺点。
  5. 建议：功能介绍，可以参考小众软件的推广文远程控制软件无限制版，VIP功能永久解锁，电脑手机平板啥都能连！。

局域网内文件互传

  1. 应用名：百灵快传
  2. 官网地址：项目主页可以下载，项目最后一次更新是在2019年，基础功能仍然很强大/
  3. 优点：
    • 操作简单。只需要在电脑端打开百灵快传，就可以在其他设备上打开浏览器进行文件传输。
    • 通用性好。拥有 Windows、macOS 与 Linux 客户端。
  4. 缺点：只能在局域网内使用。
  5. 建议：能够实现类似功能的应用还有很多，电脑和手机直接打开网页进行传输的PP直连；同样是Go语言开发的gohttpserver；以及功能丰富的CuteHttpFileServer以及可以在公网传输文件的文叔叔。

文本编辑

  1. 应用名：WPS
  2. 官网地址：WPS官网可下载
  3. 优点：
    • 免费。虽然各种office破解版横行，但是使用正版软件仍然值得鼓励。
    • 功能强大。WPS在大多数方面和Office套件可以不相上下。如果你愿意充值一个159的超级会员，还能使用完整的WPS服务（包括各种PPT、Word模板，文件翻译、文件PDF转我让的等），我个人觉得还是非常香的。
    • 性能好。尤其在处理Excel表格情境下，WPS对电脑性能的消耗低于Excel，特别适合做大量数据处理，比如计算员工工资时。
  4. 缺点：
    • 云盘同步很坑。云盘同步会让文件编辑变得卡顿。并且有些公司文件本身可能涉及商业秘密，同步行为本身就很危险，并且他还默认可以公开上传的文件，需要自己关闭这个选项。
    • 与Office套件的展示结果不一致。这在Word中展现的最频繁，所以最好使用PDF进行文件交流，文件发送和保存时注意文件后缀。
    • 会出现崩溃，与Office套件一样，莫名奇妙的就崩溃了，不过频率不多，可以接受。
  5. 建议：根据电脑性能恰当使用云盘备份能够减少文件误删的烦恼。

公文排版

  1. 应用名：小恐龙
  2. 官网地址：小恐龙官网可以下载
  3. 优点：这是一个公文排版的辅助工具，适用于政府和企业。可以自动设置文件抬头等公文格式，非常好用。
  4. 缺点：经常会出现奇怪的错误，需要关闭文件多试几次。
  5. 建议：专用插件，不写公文用处不大。

图片编辑

我觉得WPS自带的图片浏览和编辑功能超级强大，满足了我对图片处理的基础的功能需求，裁剪、制定分辨率、放缩、标注。对我来说，基本完全替代了PS的功能。

截图

日常截图

  1. 应用名：Snipaste
  2. 官网地址：官网下载即可
  3. 优点：安静且占用少，适合开机启动；功能强大且明确，功能不多但是吃透了截图交流的基本需求（截图、标注、文本解释）完全足以应对95%的情景；自从发现了F3可以题图之后，更是爱不释手。
  4. 缺点：除了快捷键经常会与Total Command冲突之外（TC实在是占用了太多Hotkey了）基本没有缺点。
  5. 建议：装机必备。

截图编辑

  1. 应用名：Share X
  2. 官网地址：官网下载即可
  3. 优点：
    • 简单图片编辑，软件提供的图标非常适合做教程；
    • 有明确的工作流，经过简单设置能够实现图片编辑与上传，非常适合Windows环境下Markdown文件写作。
  4. 缺点：工作流太长，一般的截图用不上，太繁琐。因此可以搭配一个便捷的截图软件（我常用Snipaste），或者直接使用微信自带的截图软件也够用了。
  5. 建议：由于他的功能很强大，显得有些冗余，我只有编辑博客文章时会用到。图床设置参考少数派作者FobAKQKYYX文章。快速套壳方法参见少数派作者”柯帕”的文章一日一技 | 使用 ShareX 快速为截图套壳。
  6. 配置文件：网盘/软件配置文件/Share-13.5.0

屏幕录制

  1. 应用名：Captura
  2. 官网地址：项目主页下载即可
  3. 优点：
    • 免费，完全免费，并且不限制录制时间；
    • 功能强大，录制格式种类多，能够对音、视频质量以及录制范围进行设置，满足日常需要。
  4. 缺点：录视频需要使用到FFpmeg，由于网络原因，无法使用内置方法安装FFpmeg。建议从官网下载FFpmeg，解压到制定文件夹，选择FFpmeg中的bin目录作为启动目录。
  5. 建议：。

代码编辑

  1. 应用名：Visual Studio Code
  2. 官网地址：Visual Studio Code官网可以下载，轻度使用的话，下载稳定版即可。
  3. 优点：可以编辑代码、Markdown文本，并和GitHub进行交互等功能。插件丰富，功能全面。
  4. 缺点：轻度使用，没有发现太多缺点。
  5. 建议：换电脑时候，可以使用 “setting sync”插件，进行插件设置，网上很多教程。

文本翻译

  4. 缺点：文件翻译的效果比较差，要联网。
  5. 建议：DeepL也是一款很好用的翻译软件，我觉得他不太稳定，至少在我们公司WIFI网络下。

下载器

  1. 应用名：Internet Download Manager（IDM）
  2. 官网地址：IDM官网
  3. 优点：
    • 无打扰。不用的时候完全基本看不到，需要下载的时候启动非常迅速。
    • 嗅探功能非常好用。可以自动查找到网页中可以下载的视频文件，尤其在下载视频的时候非常方便。
  4. 缺点：IDM会同时提出多个下载请求，每个下载请求下载一部分文件，最后在本地拼合。但是如果下载的文件不支持多次请求，需要在“设置”-“文件类型”中忽略这个网址，然后使用浏览器默认下载器才行。
  5. 建议：可以搭配一个迅雷5下载器，用于下载Bt链接，不过单独使用IDM也完全够用。最近发现迅雷下载器还是非常强大，普通文件下载可以完全交给迅雷。但是迅雷的各种广告弹窗让我不胜其扰。

密码管理

  1. 应用名：1password
  2. 官网地址：1password官网。我一般网页配合浏览器插件使用
  3. 优点：密码复杂难以破解，有助于提升安全性。家人共享策略不用来回问密码了
  4. 缺点：终于明白1password，为啥总有免费试用的活动了，成瘾性极强，没有了1password啥密码也不知道，也不可能记得住。使用第三方登录担心第三方获取自己的隐私，密码登录太麻烦了。
  5. 建议：虽然一直使用1password，仍然没有完全相信他。为了保险起见，所有的支付密码均没有上传到1password中。也有推荐Bitwarden，免费且跨平台协作更好，以后有机会再说吧。

文件管理

  1. 应用名：Total Command
  2. 官网地址：官网下载,张学思汉化版下载
  3. 优点：我最初的需求是“在同一个业面展示两个文件夹内容”，发现了total commander。因为想炫技，所以花了一天时间学习教程，最后发现真香，基本不用打开文件夹了。
  4. 缺点：学习成本高，推荐关注水木清华中TC板块，由于垃圾百度，善用佳软老师的教程——《一本关于 Total Commander 的书》暂时消失在互联网中，他已经决定把电子书逐步导入到语雀平台。目前可以参考老版的教程。或者在语雀平台上蹲一波TC学堂：Total Commander 电子书。

文件检索

  1. 应用名：Listary
  2. 官网地址：官网下载免费版就够用了
  3. 优点：
    • 快速启动，搭配快捷键“连击两次Ctrl”，启动非常迅速；
    • 小巧、美观，相比Everything来说，好看多了，没有太多的选项，没有使用压力。
    • 快速启动应用，输入应用名找到合适的文件就能启动应用，非常方便。
    • 自定义快捷方式，比如我经常需要到天眼查上查询公司资料，可以设置快捷方式“输入tyc就能直接进入天眼查查询信息”
  4. 缺点：没想到缺点
  5. 建议：装机必备

解压缩

  1. 应用名：7-zip
  2. 官网地址：官网可以下载，不过需要搭梯子。可以尝试中文站，我之前没有用过这个下载源，请大家自行判断。
  3. 优点：
    • 小巧，安装包只有1.4M，非常适合移动电脑。
    • 专业，除非需要压缩与解压缩，否则你不会看到它的身影。
    • 高压缩比。
  4. 缺点：唯一缺点可能是没有桌面应用吧，部分人群可能不太习惯。
  5. 建议：用不惯7-zip，也可以用360压缩，除了弹窗多，其他还好。

优化与美化

桌面壁纸

  1. 应用名：Bing Wallpaper
  2. 官网地址：https://www.microsoft.com/zh-cn/bing/bing-wallpaper-msn?pc=W069&rtc=1
  3. 优点：占用小，安装包仅有9M，基本不挑电脑。功能简单，只有更换壁纸一个功能。内容丰富，每天8张壁纸。
  4. 缺点：图片审美不一定匹配需求与场合，最好不要开启自动更新。只在合适时间更新成喜欢的图片。
  5. 建议：微软中国团队也开发可一款桌面壁纸应用，并且还有时钟、便签和待办事项功能，但是目前还在测试阶段，之后有机会再测试。关于“微软必应壁纸”参考小众软件推荐文章微软必应壁纸测试版发布，支持每日更换必应壁纸、3 个小组件、浏览器放劫持等功能。

锁屏壁纸

  1. 应用名：微软锁屏画报
  2. 获取：微软锁屏画报官网

  5. 建议：关于“微软锁屏画报”参考小众软件推荐文章微软锁屏画报 测试版上线，专门美化你的 Windows 锁屏界面。
